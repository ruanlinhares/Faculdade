# LISTA SIMPLEMENTE LIGADA COM VETOR

## As funções dessa atividade serão:
<ul>
<li>criarLista</li>
<li>inserirElemento (adiciona o elemento no final da lista)</li>
<li>inserirElementoID (adiciona o elemento na posição indicada da lista) </li>
<li>inserirElementoInicio (adiciona o elemento no início lista) </li>
<li>listarElementos (apresenta todas as informações dos elementos da lista - do primeiro ao último elemento)</li>
<li>removerElemento (remove o elemento pela posição)</li>
<li>atualizar (altera os campos de um elemento da lista)</li>
<li>buscarElemento (encontra e retorna o elemento pesquisado)</li>
<li>tamanho (retorna a quantidade de elementos da lista)</li>
<li>excluirLista (apaga todos os elementos e a lista)</li>
<li>carregarDados (realiza a leitura dos dados presentes no arquivo)</li>
<li>salvarDados (armazena os dados presentes na lista no arquivo)</li>
<li>main (usando como uma aplicação e chamando todas as funções anteriores)</li>
</ul>
